# DOTJORDAN-Rahaf

This organization contains my Full-Stack Development training projects completed during the DOTJordan training program.

## Projects Included

- Frontend projects using HTML, CSS, JavaScript, Bootstrap, React, Vite, and Next.js
- Backend projects using Node.js, Express.js, REST APIs, CRUD operations, MongoDB, MySQL, and Postman
- Full-stack projects connected between frontend and backend
- Practice projects focused on Git, GitHub, client-server communication, routing, APIs, database handling, and deployment

## Deployment

Some projects are deployed online using modern deployment platforms:

- Frontend projects are deployed using Vercel
- Backend APIs are deployed using Render
- Full-stack projects connect a deployed frontend with a deployed backend API

### Deployed Project Example

Task Manager is a full-stack project with a React frontend and a Node.js/Express backend.

- Frontend Live Demo: https://task-manager-frontend-xi-topaz.vercel.app/
- Backend API: https://taskmanager-backend-98x0.onrender.com
